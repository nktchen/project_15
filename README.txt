project for university
ТГ бот для составления погоды по маршруту
/start - показать приветственное сообщение
/help - показать список доступных команд и краткую инструкцию по использованию бота
/weather - составить маршрут и прогноз погоды!

API - AccuWeather


для запуска проекта перейдите в папку проекта (red_project_12), и введите следующую команду:

python3 -m venv venv
или python -m venv venv

source ./venv/bin/activate
pip install -r requirements.txt
это настроит виртуальную среду и установит необходимые библиотеки

потом введите команду
python3 main.py
или python main.py
